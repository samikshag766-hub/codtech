# API Integration Task

A simple web project demonstrating API integration using HTML, CSS, and JavaScript.

## Features

- **API Fetching**: Retrieve and display data from a public API
- **Responsive UI**: Clean and simple user interface
- **Vanilla JavaScript**: No frameworks required

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript

## Usage

1. Open `index.html` in your browser.
2. Interact with the page to fetch and display API data.

## Project Structure

- `index.html` – Main HTML file
- `style.css` – Stylesheet
- `script.js` – JavaScript logic for API calls
