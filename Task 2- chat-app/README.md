# Real-Time Chat App

A real-time chat application with a Node.js backend and a simple frontend.

## Features

- **Real-Time Messaging**: Chat instantly with other users
- **WebSocket Communication**: Fast, bidirectional updates
- **Simple UI**: Easy-to-use chat interface

## Tech Stack

- **Backend**: Node.js, Express, Socket.io
- **Frontend**: HTML, CSS, JavaScript

## Installation

1. Install dependencies:
   ```
   npm install
   ```
2. Start the server:
   ```
   node server.js
   ```
3. Open `public/index.html` in your browser.

## Project Structure

- `server.js` – Node.js server with Socket.io
- `public/` – Frontend files (HTML, CSS, JS)
